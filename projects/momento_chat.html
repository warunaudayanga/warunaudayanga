Momento Chat

Project Details

Chat Back-end for the Momento Mobile Application and Web Admin Front-end and also all Web Admin Front-end Chat Interfaces of Momento, An event planing platform

Functionalities
  • Single Chat
  • Group Chat
  • Group Admins
  • Send Broadcast Messages
  • Delete Messages
Used Languages

  • TypeScript
  • HTML
  • CSS
  • NoSQL
Database

Runtime Environment

Frameworks and Libraries

    • NestJS

  • Angular (Front-end)
