Learning Style Analyzer

Project Details
Info

I created a web app for a Master's degree research project that uses the Felder-Silverman Learning Styles Model to identify individual learning preferences. The goal is to help educators adapt their teaching methods, promoting a more personalized and effective learning experience for students.
