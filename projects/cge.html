Weighing Software

Project Details

Front-end and Back-end of a Weighting Software for Ceylon Grain Elevators PLC (Prima)

Functionalities
  • Read data from indicator through TCP/IP
  • Sending feedback to display through Serial Com Port
  • Scheduled Database backups to a remote Database
  • Managing weightings data
  • Generate Reports (Daily / Weekly / Monthly / Annually)
Used Languages
  • JAVA
  • HTML
  • CSS
  • JavaScript
  • SQL
Database
  • MySQL
Frameworks and Libraries
  • JAVA (Back-end)
    • Spring Boot Framework
    • JasperReports Library
  • Angular (Front-end)
    • Angular 11
    • Angular Material UI
Other Tools
  • Electron JS
    • For wrapping Web Application as a Windows Application
